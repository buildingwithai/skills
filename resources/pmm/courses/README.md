# Course Materials

Notes and transcripts from PMM courses, training, and educational content.

## Structure

Each course gets its own directory:

```
course-name/
├── README.md              # Course overview & recommendation
├── module-01-notes.md     # Detailed notes per module
├── module-02-notes.md
├── ...
└── key-takeaways.md       # Actionable insights
```

## Course Directory Template

**README.md:**
```markdown
# [Course Name]

**Instructor:** [Name]
**Platform:** [Coursera, Product Marketing Alliance, Reforge, etc.]
**Date Completed:** [Month Year]
**Recommend?** ✅ Yes / ⚠️ With caveats / ❌ No

## Course Overview

[What the course covers]

## Key Takeaways

1. [Major insight 1]
2. [Major insight 2]
3. [Major insight 3]

## Module Notes

- [module-01-notes.md](./module-01-notes.md) - [Module topic]
- [module-02-notes.md](./module-02-notes.md) - [Module topic]

## Actionable Frameworks

- **[Framework 1]**: [Brief] → Added to `resources/frameworks/`
- **[Framework 2]**: [Brief] → Applied in `[skill-name]` skill

## Would I Recommend?

**Pros:**
- [Pro 1]
- [Pro 2]

**Cons:**
- [Con 1]
- [Con 2]

**Best for:** [Who should take this course]
**Cost:** [Price or subscription]
**Time commitment:** [Hours/weeks]
```

## Adding a Course

1. Create directory: `mkdir course-name`
2. Add README with overview and recommendation
3. Add module notes as you take the course
4. Extract actionable frameworks to `resources/frameworks/`
5. Apply learnings to relevant skills
