# Research

Industry research, analysis, and best practices that inform PMM skills.

## Topics

- **messaging/** - Message testing platforms, frameworks, copywriting research
- **positioning/** - Positioning methodologies, category creation, market analysis
- **gtm/** - Go-to-market strategy, launch best practices, channel research
- **competitive-intelligence/** - Win/loss analysis, competitive frameworks, battle card research
- **pricing/** - Pricing strategy, value-based pricing, packaging research
- **customer-research/** - User interviews, JTBD, customer development

## File Format

When adding research, use this template:

```markdown
# [Title of Research]

**Source:** [Where this came from]
**Date:** [When published/accessed]
**Relevance:** [Why this matters for PMM work]

## Summary

[Key takeaways in 3-5 bullet points]

## Full Notes

[Detailed notes, organized by section]

## Application to PMM Skills

**Applies to skills:**
- [Skill name 1]: [How it relates]
- [Skill name 2]: [How it relates]

**Key frameworks/methods:**
- [What you can extract and use]

## References

- [Link to original source]
- [Related research]
```

## Naming Convention

`topic/descriptive-name-source-date.md`

**Examples:**
- `messaging/message-testing-platforms-comparison-2025.md`
- `positioning/dunford-obviously-awesome-notes-2024.md`
- `gtm/b2b-saas-launch-best-practices-2025.md`
