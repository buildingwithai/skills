# Personas Skill Assets

Templates and tools for persona creation, validation, and activation.

## Templates Directory

All templates are ready to copy and customize for your use.

### Core Templates

**1. persona-one-pager.md**
- Complete persona template (copy/paste ready)
- Includes all essential sections
- Quick reference card at bottom
- Use for final persona documentation

**2. 30-minute-interview-script.md**
- Full interview script with timing
- 5 Rings methodology integrated
- Opening, questions, closing, and post-interview tasks
- Copy and use for customer interviews

**3. proto-persona-workshop-canvas.md**
- 2-4 hour workshop template
- Individual brainstorm + group build-out
- Validation planning included
- Remote workshop instructions (Miro/Zoom)

### Validation & Testing

**4. validation-board.md**
- Track evidence for each persona attribute
- Validated ✅ / Rejected ❌ / Uncertain ⚠️
- Calculate validation score
- Quarterly review process

**5. survey-template.md**
- 20-30 question survey
- Validates all 5 Rings
- Sample size guidance (100-300 responses)
- Analysis instructions included

### Activation & Rollout

**6. activation-checklist.md**
- Week-by-week rollout plan
- Department-by-department tasks
- Systems integration checklist
- Adoption metrics to track

### Advanced

**7. negative-persona-template.md**
- Document who NOT to target
- CAC/LTV calculations
- Exclusion strategies
- Warning signs and red flags

## How to Use These Templates

### Quick Start
1. Start with **30-minute-interview-script.md** to interview customers
2. Use **persona-one-pager.md** to document findings
3. Validate with **validation-board.md** or **survey-template.md**
4. Roll out using **activation-checklist.md**

### Workshop Approach
1. Use **proto-persona-workshop-canvas.md** for rapid creation
2. Document in **persona-one-pager.md**
3. Validate with customer interviews using **30-minute-interview-script.md**
4. Track validation with **validation-board.md**

## Customization Tips

**DO customize:**
- Company-specific questions
- Industry terminology
- Persona names and roles
- Your product category language

**DON'T remove:**
- Core structure (5 Rings framework)
- Validation requirements
- Data source tracking
- Update schedules

## File Formats

All templates are in Markdown (.md) format for:
- Easy editing in any text editor
- Version control friendly
- Can convert to PDF, Word, or other formats
- Works in Notion, Confluence, etc.
