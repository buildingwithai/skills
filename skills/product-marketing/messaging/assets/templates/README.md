# Assets: Templates

Ready-to-use templates for creating product messaging and positioning.

## Available Templates

Templates will be added here as they're developed. Each template includes:
- Purpose and when to use it
- Instructions for filling it out
- Example of completed template

### Planned Templates

**Core Messaging:**
- `messaging-canvas-template.md` - Main framework for organizing all messaging
- `persona-matrix-template.md` - Map messages to different personas/audiences
- `value-proposition-template.md` - Pain-Claim-Gain structure

**Positioning:**
- `positioning-canvas-template.md` - April Dunford's 5-component framework
- `competitive-matrix-template.md` - Position against competitors

**Testing:**
- `message-testing-script.md` - Interview guide for testing messages
- `sales-feedback-template.md` - Gather input from sales team
- `messaging-audit-checklist.md` - Evaluate existing messaging

**Planning:**
- `timeline-template.md` - Gantt chart with standard durations
- `launch-checklist.md` - Everything to update during launch

**Enablement:**
- `enablement-deck-template.md` - Present messaging to sales/CS
- `battle-card-template.md` - One-pager per competitor

## Using Templates

1. Copy the template file
2. Follow the instructions in each section
3. Replace placeholder text with your content
4. Review example for reference

## Contributing Templates

When adding new templates:
- Use clear placeholder text: `[Your product name]`, `[Target audience]`
- Include filled example section
- Add usage instructions at top
- Cross-reference from SKILL.md
