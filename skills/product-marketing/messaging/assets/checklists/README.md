# Assets: Checklists

Process checklists for key activities in the positioning and messaging workflow.

## Available Checklists

Checklists will be added here as they're developed. Each checklist includes:
- When to use it
- Step-by-step items to complete
- Who should be involved

**Workshop Preparation:**
- `positioning-workshop-prep.md` - Everything needed before running positioning workshop
- `messaging-workshop-prep.md` - Setup for messaging development sessions

**Launch:**
- `internal-launch-checklist.md` - Internal rollout checklist
- `external-launch-checklist.md` - Market launch checklist
- `sales-enablement-checklist.md` - Sales team enablement

**Testing:**
- `pre-launch-testing-checklist.md` - Validate before going live
- `messaging-audit-checklist.md` - Evaluate existing messaging

**Stakeholder Management:**
- `stakeholder-buy-in-checklist.md` - Getting cross-functional buy-in
- `executive-approval-checklist.md` - Securing executive approval

## Using Checklists

1. Review the checklist before starting the activity
2. Check off items as you complete them
3. Note any blockers or dependencies
4. Verify completion before moving to next phase
