# Music Video Integration

When to call `music-video-direction` skill:

## Use Cases

1. **Song-based content:** User has music, needs video concepts
2. **Visual storytelling:** Need cinematography for product demo
3. **Editing patterns:** Matching cuts to beat/rhythm
4. **Performance content:** How to shoot artist performing

## Workflow Integration

**This Skill (Strategy):**
- Generate concept: "Bad Spanish" teacher → success story
- Script hook structure
- Identify controversy angle

**Music-Video Skill (Execution):**
- Shot list for classroom scene
- Performance footage direction
- Editing rhythm matching song tempo
- Visual metaphors for lyrics

## Handoff Point

After [03-provocative-hook-builder.md](../core/03-provocative-hook-builder.md), if music content:
→ Call `music-video-direction` with concept + script
→ Get back: Shot list, editing guide, cinematography notes
