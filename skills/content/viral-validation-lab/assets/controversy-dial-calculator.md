# Controversy Dial Calculator

## Input Your Product Type

**Professional SaaS/B2B:**
→ Dial: 60%
→ Tactics: Category attacks, expose secrets, bold decisions

**Consumer Tech/Apps:**
→ Dial: 80%
→ Tactics: Forbidden angles, absurdist humor, double meanings

**Music/Entertainment:**
→ Dial: 95%
→ Tactics: Rule-breaking, genre-bending, cultural mixing

**Taboo/Niche Products:**
→ Dial: 100%
→ Tactics: Embrace absurdity, normalize taboo, lean in fully

## Fine-Tune by Platform

**LinkedIn:** -10% (more professional)
**TikTok:** +5% (more permissive)
**Instagram:** Base level
**YouTube:** Base level

## Audience Adjustment

**Conservative industry:** -15%
**Progressive/creative industry:** +10%
**Niche community (already embraces topic):** +15%

## Final Controversy Setting

[Your base dial] + [platform adjustment] + [audience adjustment] = **Your Controversy Level**

Use this level when generating concepts in [02-controversy-content-generator.md](../references/core/02-controversy-content-generator.md)
