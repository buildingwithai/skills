# Skill Integration Flowchart

```
START: User brings idea/product/feature
    ↓
    [Q: Know your ICP?]
    NO → Use `positioning` skill (ICP Analysis) → Continue
    YES ↓
    
    [Q: Know competitive positioning?]
    NO → Use `positioning` skill (Dunford Framework) → Continue
    YES ↓
    
    [Q: Have messaging framework?]
    NO → Use `messaging` skill (Pain-Claim-Gain) → Continue
    YES ↓
    
    [Q: Want competitive content intel?]
    YES → Use `competitive-intelligence` skill → Continue
    NO ↓
    
    [Q: What do you need?]
    
    → CONTENT IDEAS:
       Use: 02-controversy-content-generator.md
       Output: 10-15 concepts
       ↓
       [Q: 10+ concepts to rank?]
       YES → Use `prioritization` skill
       NO → Use built-in scoring
       ↓
       Continue to SCRIPT
    
    → FEATURE DECISION:
       Use: 04-feature-prioritization-by-content.md
       Output: Build order
       ↓
       Continue to SCRIPT
    
    → SCRIPT CONTENT:
       Use: 03-provocative-hook-builder.md
       ↓
       [Q: Music/video content?]
       YES → Use `music-video-direction` skill
       NO ↓
       Output: Complete script
       ↓
       Continue to TEST
    
    → TEST:
       Use: 06-testing-validation-protocol.md
       Post 3 variations
       Measure 48 hours
       ↓
       [Results?]
       
       75%+ completion, 5%+ shares, good controversy
       → ✅ BUILD PRODUCT
       
       50-74% completion, mixed results
       → ⚠️ ITERATE (back to 02-generator)
       
       <50% completion, low engagement
       → ❌ ABANDON (try different idea)
```

## Quick Paths:

**Path 1: New Idea → Full Validation**
positioning → messaging → 02-generator → 03-script → 06-test
Time: 1-2 weeks

**Path 2: Existing Product → Content**
(Skip positioning/messaging if you have it)
02-generator → 03-script → 06-test
Time: 3-5 days

**Path 3: Feature Priority**
04-feature-prioritization → 03-script → 06-test
Time: 1 week
